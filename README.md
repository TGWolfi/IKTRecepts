Receptek

Csapat: Tatai Gábor (HTML és CSS), Simon Tamás (C#)

Receptgyűjtemény

A program konzolos felületen keresztül kezeli a recepteket, amelyeket objektumként tárol egy listában.
A felhasználó tud:
  új receptet hozzáadni
  keresni
  szűrni
  adatokat menteni CSV fájlba
A program különlegessége, hogy a tárolt adatokból automatikusan HTML weboldalt generál.
Ehhez egy sablont használ, amelyet a program feltölt adatokkal.

Nem jöttem rá miért is pontosan de a generált weboldal csak a (bin/debug/net8.0/output) mappán belül érhető el.
